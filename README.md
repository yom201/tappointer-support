# Tap Pointer Support Site

Static GitHub Pages site for `tappointer.com`.

## Pages

- Home: `https://tappointer.com/`
- Privacy Policy: `https://tappointer.com/privacy-policy.html`
- Privacy redirect: `https://tappointer.com/privacy`
- Terms: `https://tappointer.com/terms-of-use.html`
- Support: `https://tappointer.com/support.html`

## Deploy

Push `main` to `git@github.com:yom201/tappointer-support.git`, then enable GitHub Pages from the repository root.

DNS should point `tappointer.com` to GitHub Pages, and `CNAME` must contain only:

```txt
tappointer.com
```
